# jyothivenkat.com

Personal portfolio website for Jyothi Venkat — UX Research Leader & AI Builder.

Live at [jyothivenkat.com](https://jyothivenkat.com)

## Pages

- **Home** — Introduction and highlights
- **Work** — Case studies and project portfolio
- **Articles** — Writing and thought leadership
- **My Story** — Background and career journey
- **Connect** — Contact information

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- Google Fonts (DM Serif Display, Inter)
- SEO optimized with Open Graph and Twitter Card meta tags
- `sitemap.xml` and `robots.txt` for search indexing

## Local Development

Open `index.html` in a browser, or serve with any static file server:

```bash
python3 -m http.server 8000
```

## Structure

```
index.html          # Home page
about.html          # My Story
projects.html       # Work / Case Studies
writing.html        # Articles
connect.html        # Contact
styles.css          # All styles
script.js           # Interactions
headshot.jpg        # Profile photo
sitemap.xml         # SEO sitemap
robots.txt          # Crawler directives
```
